# my-test-server

A test MCP server

## Installation

```bash
pip install my_test_server
```

## Quick Start

```bash
# Run the MCP server (stdio mode)
python -m my_test_server.server
```

## Tools

- **echo** — Echo back messages (example tool)

## Development

```bash
pip install -e ".[dev]"
pytest
```
